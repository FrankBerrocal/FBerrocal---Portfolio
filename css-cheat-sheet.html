CSS Elements

align-content

Specifies the alignment between the lines inside a flexible container when the items do not use all available space. (W3Schools, n.d.).

align-items

Specifies the alignment for items inside a flexible container. (W3Schools, n.d.).

align-self

Specifies the alignment for selected items inside a flexible container. (W3Schools, n.d.).

all

Resets all properties (except unicode-bidi and direction). (W3Schools, n.d.).

animation

A shorthand property for all the animation-* properties. (W3Schools, n.d.).

animation-delay

Specifies a delay for the start of an animation. (W3Schools, n.d.).

animation-direction

Specifies whether an animation should be played forwards, backwards or in alternate cycles. (W3Schools, n.d.).

animation-duration

Specifies how long an animation should take to complete one cycle. (W3Schools, n.d.).

animation-fill-mode

Specifies a style for the element when the animation is not playing (before it starts, after it ends, or both). (W3Schools, n.d.).

animation-iteration-count

Specifies the number of times an animation should be played. (W3Schools, n.d.).

animation-name

Specifies a name for the @keyframes animation. (W3Schools, n.d.).

animation-play-state

Specifies whether the animation is running or paused. (W3Schools, n.d.).

animation-timing-function

Specifies the speed curve of an animation. (W3Schools, n.d.).

background

A shorthand property for all the background-* properties. (W3Schools, n.d.).

background-blend-mode

Specifies the blending mode of each background layer (color/image). (W3Schools, n.d.).

background-color

Specifies the background color of an element. (W3Schools, n.d.).

background-image

Specifies one or more background images for an element. (W3Schools, n.d.).

background-position

Specifies the position of a background image. (W3Schools, n.d.).

background-repeat

Sets if/how a background image will be repeated. (W3Schools, n.d.).

background-size

Specifies the size of the background images. (W3Schools, n.d.).

border

A shorthand property for border-width, border-style and border-color. (W3Schools, n.d.).

border-bottom

A shorthand property for border-bottom-width, border-bottom-style and border-bottom-color. (W3Schools, n.d.).

border-bottom-color

Sets the color of the bottom border. (W3Schools, n.d.).

border-bottom-style

Sets the style of the bottom border. (W3Schools, n.d.).

border-bottom-width

Sets the width of the bottom border. (W3Schools, n.d.).

border-collapse

Sets whether table borders should collapse into a single border or be separated. (W3Schools, n.d.).

border-color

Sets the color of the four borders. (W3Schools, n.d.).

border-image

A shorthand property for all the border-image-* properties. (W3Schools, n.d.).

border-image-outset

Specifies the amount by which the border image area extends beyond the border box. (W3Schools, n.d.).

border-image-repeat

Specifies whether the border image should be repeated, rounded or stretched. (W3Schools, n.d.).

border-image-slice

Specifies how to slice the border image. (W3Schools, n.d.).

border-image-source

Specifies the path to the image to be used as a border. (W3Schools, n.d.).

border-image-width

Specifies the width of the border image. (W3Schools, n.d.).

border-left

A shorthand property for all the border-left-* properties. (W3Schools, n.d.).

border-left-color

Sets the color of the left border. (W3Schools, n.d.).

border-left-style

Sets the style of the left border. (W3Schools, n.d.).

border-left-width

Sets the width of the left border. (W3Schools, n.d.).

border-radius

A shorthand property for the four border-*-radius properties. (W3Schools, n.d.).

border-right

A shorthand property for all the border-right-* properties. (W3Schools, n.d.).

border-right-color

Sets the color of the right border. (W3Schools, n.d.).

border-right-style

Sets the style of the right border. (W3Schools, n.d.).

border-right-width

Sets the width of the right border. (W3Schools, n.d.).

border-spacing

Sets the distance between the borders of adjacent cells. (W3Schools, n.d.).

border-style

Sets the style of the four borders. (W3Schools, n.d.).

border-top

A shorthand property for border-top-width, border-top-style and border-top-color. (W3Schools, n.d.).

border-top-color

Sets the color of the top border. (W3Schools, n.d.).

border-top-left-radius

Defines the radius of the border of the top-left corner. (W3Schools, n.d.).

border-top-right-radius

Defines the radius of the border of the top-right corner. (W3Schools, n.d.).

border-top-style

Sets the style of the top border. (W3Schools, n.d.).

border-top-width

Sets the width of the top border. (W3Schools, n.d.).

border-width

Sets the width of the four borders. (W3Schools, n.d.).

bottom

Sets the elements position, from the bottom of its parent element. (W3Schools, n.d.).

box-decoration-break

Sets the behavior of the background and border of an element at page-break, or, for in-line elements, at line-break. (W3Schools, n.d.).

box-shadow

Attaches one or more shadows to an element. (W3Schools, n.d.).

box-sizing

Defines how the width and height of an element are calculated: should they include padding and borders, or not. (W3Schools, n.d.).

break-after

Specifies whether or not a page-, column-, or region-break should occur after the specified element. (W3Schools, n.d.).

break-before

Specifies whether or not a page-, column-, or region-break should occur before the specified element. (W3Schools, n.d.).

break-inside

Specifies whether or not a page-, column-, or region-break should occur inside the specified element. (W3Schools, n.d.).

caption-side

Specifies the placement of a table caption. (W3Schools, n.d.).

caret-color

Specifies the color of the cursor (caret) in inputs, textareas, or any element that is editable. (W3Schools, n.d.).

@charset

Specifies the character encoding used in the style sheet. (W3Schools, n.d.).

clear

Specifies what should happen with the element that is next to a floating element. (W3Schools, n.d.).

clip

Clips an absolutely positioned element. (W3Schools, n.d.).

color

Sets the color of text. (W3Schools, n.d.).

column-count

Specifies the number of columns an element should be divided into. (W3Schools, n.d.).

column-fill

Specifies how to fill columns, balanced or not. (W3Schools, n.d.).

column-gap

Specifies the gap between the columns. (W3Schools, n.d.).

column-rule

A shorthand property for all the column-rule-* properties. (W3Schools, n.d.).

column-rule-color

Specifies the color of the rule between columns. (W3Schools, n.d.).

column-rule-style

Specifies the style of the rule between columns. (W3Schools, n.d.).

column-rule-width

Specifies the width of the rule between columns. (W3Schools, n.d.).

column-span

Specifies how many columns an element should span across. (W3Schools, n.d.).

column-width

Specifies the column width. (W3Schools, n.d.).

columns

A shorthand property for column-width and column-count. (W3Schools, n.d.).

content

Used with the :before and :after pseudo-elements, to insert generated content. (W3Schools, n.d.).

counter-increment

Increases or decreases the value of one or more CSS counters. (W3Schools, n.d.).

counter-reset

Creates or resets one or more CSS counters. (W3Schools, n.d.).

cursor

Specifies the mouse cursor to be displayed when pointing over an element. (W3Schools, n.d.).

direction

Specifies the text direction/writing direction. (W3Schools, n.d.).

display

Specifies how a certain HTML element should be displayed. (W3Schools, n.d.).

empty-cells

Specifies whether or not to display borders and background on empty cells in a table. (W3Schools, n.d.).

filter

Defines effects (e.g. blurring or color shifting) on an element before the element is displayed. (W3Schools, n.d.).

flex

A shorthand property for the flex-grow, flex-shrink, and the flex-basis properties. (W3Schools, n.d.).

flex-basis

Specifies the initial length of a flexible item. (W3Schools, n.d.).

flex-direction

Specifies the direction of the flexible items. (W3Schools, n.d.).

flex-flow

A shorthand property for the flex-direction and the flex-wrap properties. (W3Schools, n.d.).

flex-grow

Specifies how much the item will grow relative to the rest. (W3Schools, n.d.).

flex-shrink

Specifies how the item will shrink relative to the rest. (W3Schools, n.d.).

flex-wrap

Specifies whether the flexible items should wrap or not. (W3Schools, n.d.).

float

Specifies whether an element should float to the left, right, or not at all. (W3Schools, n.d.).

font

A shorthand property for the font-style, font-variant, font-weight, font-size/line-height, and the font-family properties. (W3Schools, n.d.).

@font-face

A rule that allows websites to download and use fonts other than the "web-safe" fonts. (W3Schools, n.d.).

font-family

Specifies the font family for text. (W3Schools, n.d.).

font-feature-settings

Allows control over advanced typographic features in OpenType fonts. (W3Schools, n.d.).

@font-feature-values

Allows authors to use a common name in font-variant-alternate for feature activated differently in OpenType. (W3Schools, n.d.).

font-kerning

Controls the usage of the kerning information (how letters are spaced). (W3Schools, n.d.).

font-language-override

Controls the usage of language-specific glyphs in a typeface. (W3Schools, n.d.).

font-size

Specifies the font size of text. (W3Schools, n.d.).

font-size-adjust

Preserves the readability of text when font fallback occurs. (W3Schools, n.d.).

font-stretch

Selects a normal, condensed, or expanded face from a font family. (W3Schools, n.d.).

font-style

Specifies the font style for text. (W3Schools, n.d.).

font-synthesis

Controls which missing typefaces (bold or italic) may be synthesized by the browser. (W3Schools, n.d.).

font-variant

Specifies whether or not a text should be displayed in a small-caps font. (W3Schools, n.d.).

font-variant-alternates

Controls the usage of alternate glyphs associated to alternative names defined in @font-feature-values. (W3Schools, n.d.).

font-variant-caps

Controls the usage of alternate glyphs for capital letters. (W3Schools, n.d.).

font-variant-east-asian

Controls the usage of alternate glyphs for East Asian scripts (e.g Japanese and Chinese). (W3Schools, n.d.).

font-variant-ligatures

Controls which ligatures and contextual forms are used in textual content of the elements it applies to. (W3Schools, n.d.).

font-variant-numeric

Controls the usage of alternate glyphs for numbers, fractions, and ordinal markers. (W3Schools, n.d.).

font-variant-position

Controls the usage of alternate glyphs of smaller size positioned as superscript or subscript regarding the baseline of the font. (W3Schools, n.d.).

font-weight

Specifies the weight of a font. (W3Schools, n.d.).

gap

A shorthand property for the row-gap and the column-gap properties. (W3Schools, n.d.).

grid

A shorthand property for the grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns, and the grid-auto-flow properties. (W3Schools, n.d.).

grid-area

Either specifies a name for the grid item, or this property is a shorthand property for the grid-row-start, grid-column-start, grid-row-end, and grid-column-end properties. (W3Schools, n.d.).

grid-auto-columns

Specifies a default column size. (W3Schools, n.d.).

grid-auto-flow

Specifies how auto-placed items are inserted in the grid. (W3Schools, n.d.).

grid-auto-rows

Specifies a default row size. (W3Schools, n.d.).

grid-column

A shorthand property for the grid-column-start and the grid-column-end properties. (W3Schools, n.d.).

grid-column-end

Specifies where to end the grid item. (W3Schools, n.d.).

grid-column-gap

Specifies the size of the gap between columns. (W3Schools, n.d.).

grid-column-start

Specifies where to start the grid item. (W3Schools, n.d.).

grid-gap

A shorthand property for the grid-row-gap and grid-column-gap properties. (W3Schools, n.d.).

grid-row

A shorthand property for the grid-row-start and the grid-row-end properties. (W3Schools, n.d.).

grid-row-end

Specifies where to end the grid item. (W3Schools, n.d.).

grid-row-gap

Specifies the size of the gap between rows. (W3Schools, n.d.).

grid-row-start

Specifies where to start the grid item. (W3Schools, n.d.).

grid-template

A shorthand property for the grid-template-rows, grid-template-columns and grid-areas properties. (W3Schools, n.d.).

grid-template-areas

Specifies how to display columns and rows, using named grid items. (W3Schools, n.d.).

grid-template-columns

Specifies the size of the columns, and how many columns in a grid layout. (W3Schools, n.d.).

grid-template-rows

Specifies the size of the rows in a grid layout. (W3Schools, n.d.).

hanging-punctuation

Specifies whether a punctuation character may be placed outside the line box. (W3Schools, n.d.).

height

Sets the height of an element. (W3Schools, n.d.).

hyphens

Sets how to split words to improve the layout of paragraphs. (W3Schools, n.d.).

image-rendering

Specifies the type of algorithm to use for image scaling. (W3Schools, n.d.).

@import

Allows you to import a style sheet into another style sheet. (W3Schools, n.d.).

isolation

Defines whether an element must create a new stacking content. (W3Schools, n.d.).

justify-content

Specifies the alignment between the items inside a flexible container when the items do not use all available space. (W3Schools, n.d.).

@keyframes

Specifies the animation code. (W3Schools, n.d.).

left

letter-spacing

line-break

list-style-image

Specifies an image as the list-item marker. (W3Schools, n.d.).

list-style-position

Specifies the position of the list-item markers (bullet points). (W3Schools, n.d.).

list-style-type

margin-bottom

Sets the bottom margin of an element. (W3Schools, n.d.).

margin-left

Sets the left margin of an element. (W3Schools, n.d.).

margin-right

Sets the right margin of an element. (W3Schools, n.d.).

margin-top

Sets the top margin of an element. (W3Schools, n.d.).

Hides parts of an element by masking or clipping an image at specific places. (W3Schools, n.d.).

mask-clip

Specifies the mask area. (W3Schools, n.d.).

mask-composite

Represents a compositing operation used on the current mask layer with the mask layers below it. (W3Schools, n.d.).

mask-image

Specifies an image to be used as a mask layer for an element. (W3Schools, n.d.).

mask-mode

Specifies whether the mask layer image is treated as a luminance mask or as an alpha mask. (W3Schools, n.d.).

mask-origin

Specifies the origin position (the mask position area) of a mask layer image. (W3Schools, n.d.).

mask-position

Sets the starting position of a mask layer image (relative to the mask position area). (W3Schools, n.d.).

mask-repeat

Specifies how the mask layer image is repeated. (W3Schools, n.d.).

mask-size

Specifies the size of a mask layer image. (W3Schools, n.d.).

mask-type

Specifies whether an SVG element is treated as a luminance mask or as an alpha mask. (W3Schools, n.d.).

max-height

Sets the maximum height of an element. (W3Schools, n.d.).

max-width

Sets the maximum width of an element. (W3Schools, n.d.).

outline-color

outline-style

Sets the style of an outline. (W3Schools, n.d.).
