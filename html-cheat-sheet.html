HTML Cheat Sheet

Main elements for HTML have been included as reference on this page. Literal definitions are required since the document is an academic source for future reference for the student.

Source: W3Schools

HTML tags

<!DOCTYPE>

Defines the document type. (W3Schools, n.d.).

<html>

Defines an HTML document. (W3Schools, n.d.).

<head>

Contains metadata/information for the document. (W3Schools, n.d.).

<title>

Defines a title for the document. (W3Schools, n.d.).

<body>

Defines the document's body. (W3Schools, n.d.).

<h1> to <h6>

Defines HTML headings. (W3Schools, n.d.).

<p>

Defines a paragraph. (W3Schools, n.d.).

<br>

Inserts a single line break. (W3Schools, n.d.).

<!--...-->

Defines a comment. (W3Schools, n.d.).

Format tagsHTML

<address>

Defines contact information for the author/owner of a document/article. (W3Schools, n.d.).

<b>

Defines bold text. (W3Schools, n.d.).

<blockquote>

Defines a section that is quoted from another source. (W3Schools, n.d.).

<cite>

Defines the title of a work. (W3Schools, n.d.).

<code>

Defines a piece of computer code. (W3Schools, n.d.).

<em>

Defines emphasized text. (W3Schools, n.d.).

<font>

Defines font, color, and size for text. CSS must be used. (W3Schools, n.d.).

<small>

Defines smaller text. (W3Schools, n.d.).

<sub>

Defines subscripted text. (W3Schools, n.d.).

<sup>

Defines superscripted text. (W3Schools, n.d.).

Forms and Input

<form>

Defines an HTML form for user input. (W3Schools, n.d.).

<input>

Defines an input control. (W3Schools, n.d.).

<textarea>

Defines a multiline input control (text area). (W3Schools, n.d.).

<button>

Defines a clickable button. (W3Schools, n.d.).

<select>

Defines a drop-down list. (W3Schools, n.d.).

<optgroup>

Defines a group of related options in a drop-down list. (W3Schools, n.d.).

<option>

Defines an option in a drop-down list. (W3Schools, n.d.).

<label>

Defines a label for an <input> element. (W3Schools, n.d.).

<fieldset>

Groups related elements in a form. (W3Schools, n.d.).

<legend>

Defines a caption for a <fieldset> element. (W3Schools, n.d.).

<datalist>

Specifies a list of pre-defined options for input controls. (W3Schools, n.d.).

<output>

Defines the result of a calculation. (W3Schools, n.d.).

Frames

<iframe>

Defines an inline frame. (W3Schools, n.d.).

Images

<img>

Defines an image. (W3Schools, n.d.).

<map>

Defines a client-side image map. (W3Schools, n.d.).

<area>

Defines an area inside an image map. (W3Schools, n.d.).

<canvas>

Used to draw graphics, on the fly, via scripting (usually JavaScript). (W3Schools, n.d.).

<figcaption>

Defines a caption for a <figure> element. (W3Schools, n.d.).

<figure>

Specifies self-contained content. (W3Schools, n.d.).

<picture>

Defines a container for multiple image resources. (W3Schools, n.d.).

<svg>

Defines a container for SVG graphics. (W3Schools, n.d.).

Audio and Video

<audio>

Defines sound content. (W3Schools, n.d.).

<source>

Defines the file source. (W3Schools, n.d.).

<track>

Defines text tracks for media elements. (W3Schools, n.d.).

<video>

Defines a video or movie. (W3Schools, n.d.).

Links

<a>

Defines a hyperlink. (W3Schools, n.d.).

<link>

Defines the relationship between a document and an external resource (most used to link to style sheets). (W3Schools, n.d.).

<nav>

Defines navigation links. (W3Schools, n.d.).

Lists

<ul>

Defines an unordered list. (W3Schools, n.d.).

<ol>

Defines an ordered list. (W3Schools, n.d.).

<li>

Defines a list item. (W3Schools, n.d.).

<dl>

Defines a description list. (W3Schools, n.d.).

<dt>

Defines a term/name in a description list. (W3Schools, n.d.).

<dd>

Defines a description of a term/name in a description list. (W3Schools, n.d.).

Tables

<table>

Defines a table. (W3Schools, n.d.).

<caption>

Defines a table caption. (W3Schools, n.d.).

<th>

Defines a header cell in a table. (W3Schools, n.d.).

<tr>

Defines a row in a table. (W3Schools, n.d.).

<td>

Defines a cell in a table. (W3Schools, n.d.).

<thead>

Groups the header content in a table. (W3Schools, n.d.).

<tbody>

Groups the body content in a table. (W3Schools, n.d.).

<tfoot>

Groups the footer content in a table. (W3Schools, n.d.).

<col>

Specifies column properties for each column within a <colgroup> element. (W3Schools, n.d.).

<colgroup>

Specifies a group of one or more columns in a table for formatting. (W3Schools, n.d.).

Styles and Semantics

<style>

Defines style information for a document. (W3Schools, n.d.).

<div>

Defines a section in a document. (W3Schools, n.d.).

<span>

Defines a section in a document. (W3Schools, n.d.).

<header>

Defines a header for a document or section. (W3Schools, n.d.).

<footer>

Defines a footer for a document or section. (W3Schools, n.d.).

<main>

Specifies the main content of a document. (W3Schools, n.d.).

<section>

Defines a section in a document. (W3Schools, n.d.).

<article>

Defines an article. (W3Schools, n.d.).

<aside>

Defines content aside from the page content. (W3Schools, n.d.).

<details>

Defines additional details that the user can view or hide. (W3Schools, n.d.).

<dialog>

Defines a dialog box or window. (W3Schools, n.d.).

<summary>

Defines a visible heading for a <details> element. (W3Schools, n.d.).

<data>

Adds a machine-readable translation of a given content. (W3Schools, n.d.).

Meta Info

<head>

Defines information about the document. (W3Schools, n.d.).

<meta>

Defines metadata about an HTML document. (W3Schools, n.d.).

<base>

Specifies the base URL/target for all relative URLs in a document. (W3Schools, n.d.).

Programming

<script>

Defines a client-side script. (W3Schools, n.d.).

<noscript>

Defines an alternate content for users that do not support client-side scripts. (W3Schools, n.d.).

<embed>

Defines a container for an external (non-HTML) application. (W3Schools, n.d.).

<object>

Defines an embedded object. (W3Schools, n.d.).

<param>

Defines a parameter for an object. (W3Schools, n.d.).

Reference

W3Schools. (n.d.). HTML Element Reference - By Category. W3Schools. Retrieved on December 8th, 2021, from https://www.w3schools.com/tagsHTML/ref_byfunc.asp
